---
name: Never skip or defer tasks
description: When user gives a task, do it immediately — don't defer, skip, or suggest doing it later
type: feedback
---

Never skip or defer a task the user assigns. Don't say "let me note this and do it later" or "I'll tackle that after X." Do what the user says in the order they say it.

**Why:** The user sets priorities. Deferring tasks without permission is disrespectful of their judgment.

**How to apply:** When the user gives an instruction, execute it immediately. If it requires significant work, explain the scope and start — don't postpone.
